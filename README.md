# Advanced Concrete Design Tool 2026.05

Static browser-based concrete design prototype for beam, one-way slab strip, and two-way slab panel checks.

## Open locally

Open `index.html` in a browser, or serve the folder with:

```sh
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/index.html`.

## Notes

This is an engineering design aid and does not replace independent verification against current project requirements and licensed design standards.
